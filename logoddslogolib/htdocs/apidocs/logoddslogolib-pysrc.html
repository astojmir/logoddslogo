 48  """ 
 49  LogOddsLogo (http://www.ncbi.nlm.nih.gov/CBBresearch/Yu/logoddslogo/) is a tool 
 50  for creating sequence logos from biological sequence alignments. Building on 
 51  the WebLogo (http://weblogo.threeplusone.com) source code, LogOddsLogo uses 
 52  per-observation multiple-alignment log-odds scores as measures of information 
 53  content at each position of a sequence logo. 
 54   
 55  It can be run on the command line, as a standalone webserver, as a CGI webapp, 
 56  or as a python library. 
 57   
 58  Please consult the manual for installation instructions and more information: 
 59  htdocs/manual.html. The manual is also located at 
 60  http://www.ncbi.nlm.nih.gov/CBBresearch/Yu/logoddslogo/manual.html. 
 61   
 62  For help on the command line interface run 
 63      ./logoddslogo --help 
 64   
 65  To build a simple logo run 
 66      ./logoddslogo  < cap.fa > logo0.eps 
 67   
 68  To run as a standalone webserver at localhost:8080 
 69      ./logoddslogo --serve 
 70   
 71  To create a logo in python code: 
 72      >>> from logoddslogolib import * 
 73      >>> fin = open('cap.fa') 
 74      >>> seqs = read_seq_data(fin) 
 75      >>> data = LogoData.from_seqs(seqs) 
 76      >>> options = LogoOptions() 
 77      >>> options.title = "A Logo Title" 
 78      >>> format = LogoFormat(data, options) 
 79      >>> fout = open('cap.eps', 'w') 
 80      >>> eps_formatter( data, format, fout)
