 90  # The Lanczos approximation for the gamma function is 
 91  #                           
 92  #                -(z + g + 1/2)               (z + 1/2)                    
 93  # Gamma(z+1) =  e               * (z + g + 1/2)        * Sqrt(2Pi) * C  
 94  #                                                                         
 95  # 
 96  #                      c[1]    c[2]    c[3] 
 97  #          C = [c[0] + ----- + ----- + ----- + ...   ] 
 98  #                      z + 1   z + 2   z + 3  
 99  # 
100  # 
101  #  To calculate digamma and trigamma functions we take an analytic derivative 
102  #  of the Lanczos approximation. 
103  # 
104  #  Gamma(z)  = Gamma(z+1)/z 
105  #  Digamma(z) = D ln Gamma(z) 
106  #  Trigamma(z) = D Digamma(z)          
107   
108  # These Lanczos constants are from 
109  # "A note on the computation of the convergent 
110  # Lanczos complex Gamma approximation." Paul Godfrey (2001) 
111  # http://my.fit.edu/~gabdo/gamma.txt 
112
