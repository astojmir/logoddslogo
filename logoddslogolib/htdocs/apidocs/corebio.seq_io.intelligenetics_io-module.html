Module intelligenetics_io

source code

Read and write sequence information in IntelliGenetics format.

A sequence file in IG format can contain several sequences, each consisting of a number of comment lines that must begin with a semicolon (";"), a line with the sequence name and the sequence itself terminated with the termination character '1' for linear or '2' for circular sequences. The termination character is defacto optional.
