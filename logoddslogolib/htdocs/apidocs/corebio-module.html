Submodules
  • corebio._version
  • corebio.data: Standard data used in computational biology.
  • corebio.db: Database access.
    • corebio.db.astral: ASTRAL: Compendium for Sequence and Structure Analysis.
    • corebio.db.scop: SCOP: Structural Classification of Proteins.
  • corebio.matrix: Arrays indexed by alphabetic strings.
  • corebio.moremath: Various bits of useful math not in the standard python library.
  • corebio.secstruc: Protein secondary structure and associated tools and data.
    • corebio.secstruc.dssp: Command and control of the program DSSP: Dictionary of protein secondary structure.
    • corebio.secstruc.stride: Command and control of the program STRIDE: Protein secondary structure assignment from atomic coordinates.
  • corebio.seq: Alphabetic sequences and associated tools and data.
  • corebio.seq_io: Sequence file reading and writing.
    • corebio.seq_io._nexus: Parse the contents of a nexus file.
      • corebio.seq_io._nexus.Nodes
      • corebio.seq_io._nexus.Trees
    • corebio.seq_io.array_io: Read and write a rectangular array of sequence data.
    • corebio.seq_io.clustal_io: Read and write the CLUSTAL sequence file format.
    • corebio.seq_io.fasta_io: Read and write sequence information in FASTA format.
    • corebio.seq_io.genbank_io: Read GenBank flat files.
    • corebio.seq_io.intelligenetics_io: Read and write sequence information in IntelliGenetics format.
    • corebio.seq_io.msf_io: Read sequence information in MSF format.
    • corebio.seq_io.nbrf_io: Sequence IO for NBRF/PIR format.
    • corebio.seq_io.nexus_io: Read the sequence data from a nexus file.
    • corebio.seq_io.null_io: Null sequence IO.
    • corebio.seq_io.phylip_io: Reads Sequences in interleaved Phylip format (not sequential) and returns a list of sequences.
    • corebio.seq_io.plain_io: Read and write raw, unformatted sequence data.
    • corebio.seq_io.stockholm_io: Read a multiple sequence alignment in STOCKHOLM format.
    • corebio.seq_io.table_io: Read and write sequence information in tab-delimited format.
  • corebio.transform: Transformations of Seqs (alphabetic sequences).
  • corebio.utils: Extra utilities and core classes not in standard python.
    • corebio.utils._which: Find the full path to commands.
    • corebio.utils.deoptparse: Custom extensions to OptionParse for parsing command line options.
